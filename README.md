# Test-Aider - API REST avec Vibe Coding Local

Une API REST Node.js/Express construite 100% localement avec **Aider** + **Ollama** + **Qwen2.5**.

## 🚀 Fonctionnalités

- **GET /health** - Vérifie que l'API est opérationnelle
- **POST /scan** - Scanne un port sur une cible donnée

## 🛠️ Stack Technique

- **Backend** : Node.js + Express
- **AI Assistant** : Aider CLI + Ollama (Qwen2.5 local)
- **Versionning** : Git + GitHub
- **100% Local** : Aucune dépendance cloud, zéro coût

## 📦 Installation

```bash
# Cloner le repo
git clone https://github.com/ChromeMerc/test-aider.git
cd test-aider

# Installer les dépendances
npm install

# Lancer le serveur
npm start
